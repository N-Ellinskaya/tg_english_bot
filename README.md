# English Learning Telegram Bot

Телеграм-бот для изучения английского языка с интерактивным интерфейсом и системой уровней сложности.

## Описание
Бот помогает пользователям изучать английский язык через чтение текстов разной сложности. Пользователи могут выбирать тексты, просматривать переводы и выделять слова для изучения.

## Основной функционал
- **Система уровней сложности**: Easy, Medium, Hard
- **Интерактивное меню**: Навигация через кнопки InlineKeyboard
- **Работа с текстами**: 
  - Чтение оригинальных английских текстов
  - Просмотр профессиональных переводов
  - Выделение слов и фраз для изучения
- **Conversation Handler**: Управление сложными диалоговыми сценариями

## Технологический стек
- Python 3.x
- python-telegram-bot
- Telegram Bot API
- InlineKeyboard для UX

## Примеры текстов
Бот содержит коллекцию текстов от простых цитат до сложных предложений:
- **Easy**: Знаменитые цитаты из фильмов и книг
- **Medium**: Повседневные ситуации и диалоги  
- **Hard**: Научные и деловые тексты
